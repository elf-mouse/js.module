# js-modules-examples
Example of JavaScript modules and related tools

This is how we all start writing JavaScript.

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>JS Modules</title>
  </head>
  <body>
    <!-- Your HTML Markup goes here -->
    <script type="text/javascript">
        /* Your Javascript Code goes here! */
    </script>
  </body>
</html>

```
